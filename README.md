# mpsp-pwa

Angular Application who scrap more than 10 portals from government in a single search, optimizing searches and facilitating results.

# Running
Install Node and Npm, https://nodejs.org/en/

On root folder run <br>
```$ npm i```<br>
```$ npm run start```
